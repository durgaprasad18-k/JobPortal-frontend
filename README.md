# Job Portal Application

A full-stack Job Portal web application where users can register, login, create profiles, and apply for jobs.

## Tech Stack
- Frontend: React, TypeScript
- Backend: Spring Boot (Java)
- Database: MongoDB Atlas
- Authentication: JWT

## Features
- User Registration & Login
- JWT Authentication
- User Profile Management
- Job Listings
- Job Search & Filters
